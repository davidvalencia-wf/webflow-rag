## [Multi-Item Publish/Unpublish](https://developers.webflow.com/data/changelog/multi-item-publishunpublish)

## Added support for publishing and removing multiple Collection Items

The Data API now supports the following endpoints for bulk publishing actions:

- [Publish Collection Items](https://developers.webflow.com/data/reference/cms/collection-items/staged-items/publish-item)
- [Delete Collection Items](https://developers.webflow.com/data/reference/cms/collection-items/staged-items/delete-items)

## Webflow SDK Update to 0.8.1

The [Webflow Node.js SDK](https://github.com/webflow/js-webflow-api) is versioned to `0.8.1`, which includes methods to support multi-item publish and removal.
