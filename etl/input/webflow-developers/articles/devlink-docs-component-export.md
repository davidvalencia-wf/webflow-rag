With DevLink, you can export Webflow designs directly to React. Designers create reusable components visually in Webflow, and developers use DevLink to bring those components into React projects, including Webflow Cloud apps, with ease.

![](https://dhygzobemt712.cloudfront.net/Icons/Dark/64px/Favicon.svg)![](https://dhygzobemt712.cloudfront.net/Icons/Light/64px/Favicon.svg)

Design visually

Create components using Webflow’s visual editor with no code required.

![](https://dhygzobemt712.cloudfront.net/Icons/Dark/64px/ComponentsCode.svg)![](https://dhygzobemt712.cloudfront.net/Icons/Light/64px/ComponentsCode.svg)

Export to React

Transform Webflow components into React components with a single command.

![](https://dhygzobemt712.cloudfront.net/Icons/Dark/64px/Styles.svg)![](https://dhygzobemt712.cloudfront.net/Icons/Light/64px/Styles.svg)

Preserve design systems

Preserve styles, variables, components and interactions from your Webflow design.

Update your design system in Webflow, then sync changes to your codebase.

## Working with Exported Components

DevLink translates your Webflow components into React components through a simple workflow:

[1](https://developers.webflow.com/devlink/docs/component-export#design-in-webflow)

### Design in Webflow

Create styles, variables, and components in Webflow’s best-in-class visual editor.

[2](https://developers.webflow.com/devlink/docs/component-export#configure-devlink)

### Configure DevLink

[Set up the connection](https://developers.webflow.com/devlink/docs/component-export/installation) between your Webflow project and your React application.

[3](https://developers.webflow.com/devlink/docs/component-export#sync-your-design-system)

### Sync your design system

Use the Webflow CLI to export and [sync styles, variables, and components](https://developers.webflow.com/devlink/docs/component-export/whats-exported) to your project.

[4](https://developers.webflow.com/devlink/docs/component-export#use-your-design-system-in-your-react-app)

### Use your design system in your React app

Import the global styles and generated React components into your application code.

[5](https://developers.webflow.com/devlink/docs/component-export#add-functionality)

### Add functionality

Enhance the components with [custom logic, state, and interactivity.](https://developers.webflow.com/devlink/usage/framework-guides)
