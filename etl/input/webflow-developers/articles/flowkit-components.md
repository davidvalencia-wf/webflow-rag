# Components

Flowkit provides a comprehensive set of component classes (button, card, etc) that help you build consistent and beautiful user interfaces. Each component is designed with accessibility, customization, and ease of use in mind.

## Available Components

### Interactive Elements

- [Buttons](https://developers.webflow.com/flowkit/components/buttons) \- Primary, secondary, and tertiary button styles
- [Tag](https://developers.webflow.com/flowkit/components/tag) \- Label and category indicators
- [Dropdown](https://developers.webflow.com/flowkit/components/dropdown) \- Expandable selection menus
- [Accordion](https://developers.webflow.com/flowkit/components/accordion) \- Collapsible content sections

### Content Display

- [Images](https://developers.webflow.com/flowkit/components/images) \- Image display and responsive behavior
- [Cards](https://developers.webflow.com/flowkit/components/cards) \- Content containers and layouts
- [Rich Text](https://developers.webflow.com/flowkit/components/richtext) \- Text formatting and styles
- [Divider](https://developers.webflow.com/flowkit/components/divider) \- Section separators

### Form Elements

- [Forms](https://developers.webflow.com/flowkit/components/forms) \- Input fields, labels, and form layouts

### Navigation

- [Tabs](https://developers.webflow.com/flowkit/components/tabs) \- Content organization and navigation
- [Slider](https://developers.webflow.com/flowkit/components/slider) \- Carousel and content sliders
