## [Manage links in the Designer](https://developers.webflow.com/designer/072524)

In this release, the Designer API includes methods to manage links in the Designer.

- **[Get Link Target](https://developers.webflow.com/designer/reference/link-element/getTarget):** Get the target value of the link block element.
- **[Set Link Block Settings](https://developers.webflow.com/designer/reference/link-element/setSettings):** Apply settings for a Link Block element. Including the type of link, its value, and metadata settings.
