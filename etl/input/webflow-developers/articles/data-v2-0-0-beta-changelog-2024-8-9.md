## [Support for robots.txt](https://developers.webflow.com/data/v2.0.0-beta/changelog/2024/8/9)

In this latest release, we’ve added new endpoints to the Data API giving you more tools to access site configurations, and new methods to the Designer API, allowing you to determine a user’s permissions and capabilities in the Webflow Designer.

### New endpoints

- **[Get robots.txt](https://developers.webflow.com/data/v2.0.0-beta/reference/enterprise/site-configuration/robots-txt/get):** Retrieve the robots.txt configuration for various user agents.
