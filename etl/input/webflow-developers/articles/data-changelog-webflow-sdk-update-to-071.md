## [Webflow SDK update to 0.7.1](https://developers.webflow.com/data/changelog/webflow-sdk-update-to-071)

A significant SDK update that includes updates to all dependencies, [fixed typings](https://github.com/webflow/js-webflow-api/pull/49), and an [error message typo for `item()`](https://github.com/webflow/js-webflow-api/pull/35).

Learn more about the release on [GitHub](https://github.com/webflow/js-webflow-api/releases).
